◆ CLI_Action

typedef int(* CLI_Action) (unsigned int argument_count, const char **arguments, void *settings)

Function type for CLI actions.

A CLI_Action is called when a corresponding command-line argument is matched. It receives the count and values of arguments, as well as a pointer to user-defined settings.

Parameters
argument_countNumber of arguments passed to this action.
argumentsArray of argument strings.
settingsUser-defined settings pointer passed to the parser.
Returns
0 on success, non-zero error code on failure.
