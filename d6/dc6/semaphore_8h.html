◆ semaphore_signal()

int semaphore_signal ( struct Semaphore *  s)

Increments the semaphore count and wakes a waiting thread, if any.

Increments the semaphore count by one, making an additional unit of the resource available. If one or more threads are blocked waiting on the semaphore, one waiting thread is woken.

The calling thread does not need to have previously waited on the semaphore.

Parameters
sPointer to the semaphore.
Returns
  • 0: Semaphore successfully signaled.
  • DESCENT_ERROR_NULL: s is NULL.
  • DESCENT_ERROR_OVERFLOW: Semaphore count would exceed its maximum value.

◆ semaphore_trywait()

int semaphore_trywait ( struct Semaphore *  s)

Attempts to decrement the semaphore count without blocking.

If the semaphore count is greater than zero, it is decremented and the function returns immediately. If the count is zero, the function returns without blocking.

Parameters
sPointer to the semaphore.
Returns
  • 0: Semaphore successfully acquired.
  • THREAD_INFO_BUSY: Semaphore count is zero.
  • DESCENT_ERROR_NULL: s is NULL.

◆ semaphore_wait()

int semaphore_wait ( struct Semaphore *  s)

Decrements the semaphore count, blocking if necessary.

If the semaphore count is greater than zero, it is decremented and the function returns immediately. If the count is zero, the calling thread blocks until another thread signals the semaphore.

This function may block indefinitely.

Parameters
sPointer to the semaphore.
Returns
  • 0: Semaphore successfully acquired.
  • DESCENT_ERROR_NULL: s is NULL.
