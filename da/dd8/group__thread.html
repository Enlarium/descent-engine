Detailed Description

Threading and synchronization primitives.

This module provides low-level threading and synchronization facilities, including thread management, mutual exclusion, condition variables, semaphores, and queue-based locks.

All mechanisms in this module are intra-process only and are not safe for use across process boundaries.
